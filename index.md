Software engineering in 2026 increasingly looks like directing AI agents rather than writing every line yourself. But doing it well requires new skills: knowing how to decompose problems, when to trust agent output, and how to build workflows that actually ship.

In this workshop, attendees roll up their sleeves with **Claude Code**, Anthropic's terminal-native agentic coding tool. Hands-on exercises cover codebase navigation, task orchestration, prompt structuring, and debugging agent output. For those ready to go deeper: multi-agent patterns — running parallel Claude sessions, coordinating agent teams, and building subagent pipelines for code review and testing.
